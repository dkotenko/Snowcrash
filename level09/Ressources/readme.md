В папке исполняемый файл, который требует аргумент - файл со строкой.
Код исполняемого файла читает строку из файла-аргумента и увеличивает значение каждого символа на значение позиции символа:
	aaaaaa -> abcdef

Исползуем код на Python для декодирования токена:
	f = open('token', 'r')
	s = f.read()[:-1]
	p = [chr(ord(x) - i) for x,i in zip(list(s), range(len(s)))]
	print ''.join(p)
